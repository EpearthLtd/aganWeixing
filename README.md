# 阿甘学车微信小程序（学员端）
这是一款用于学车平台为学员提供学车报名、预约练车服务的微信小程序
This is a mini program project of WeChat

## 技术参数
* 调试基础库：2.1.1
* 业务域名：aganxueche.com | www.aganxueche.com

## 版本说明
* V1 静态海报广告：只包含广告展示和客服模块，没有约车功能
* V2 预约练车：包含预约练车的功能模块（开发中）

## 命名书写方法
* 小驼峰：页面文件命名、变量命名、参数命名
* 短横线：wxss命名、资源文件命名
