【命名规则】

自定义名称使用驼峰命名或使用“-”分隔单词

图片命名：   {name}-{style}.{type}
    name：   图片名称
    style：  图片风格    -a：active   -d：dark
    type：   文件格式
例如： logo.png    form-a.png  form-d.png
